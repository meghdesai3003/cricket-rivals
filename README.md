# Cricket Rivals

> A premium full-stack cricket trading card game built with modern web technologies.

---

## About

Cricket Rivals is a modern cricket trading card game inspired by the excitement of collectible sports cards. Players build their dream Playing XI, collect player cards, open packs, compete against AI opponents, and progress through an engaging strategy-based gameplay experience.

This project is being built as a flagship portfolio project to showcase modern full-stack software development, UI/UX design, game mechanics, and scalable application architecture.

---

## Vision

The vision of Cricket Rivals is to create a polished, immersive, and rewarding cricket card game that combines strategic gameplay with premium design and smooth user experience.

The project focuses on:

- Strategic card battles
- Collectible player cards
- Exciting pack opening system
- AI-powered gameplay
- Player progression
- Rewarding collection experience
- Premium animations and modern UI
